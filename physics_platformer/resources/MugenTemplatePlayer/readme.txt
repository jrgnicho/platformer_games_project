M.U.G.E.N Template Character
----------------------------
Base character made by Elecbyte
Template By N64Mario (N64Mario84@hotmail.com)
Updated: 12/10/05

You can use this to start your own character.
Everything is taken out except for the required lines
and the normal punches/kicks and throwing.

The sprite file included in this package includes the
required sprite set needed according to M.U.G.E.N.'s
debug information.

This template is used only for the new version of M.U.G.E.N.
